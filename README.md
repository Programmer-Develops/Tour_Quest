# Tour Quest

**Explore. Share. Inspire.**  
A full-stack **tour guide platform** where travelers share real experiences, hidden gems, and personal stories.

---

### Live Demo  
[https://tour-quest.vercel.app](https://tour-quest.vercel.app)  

---

### Tech Stack

| Layer       | Technology                     |
|-------------|--------------------------------|
| Frontend    | React.js + Material UI         |
| Backend     | Node.js + Express.js           |
| Database    | MongoDB Atlas                  |
| Auth        | Google OAuth 2.0               |
| Deployment  | Vercel (Frontend) + Render (API) |

---

### Features

- **Google Login** – Secure, fast authentication  
- **Create & Share Tours** – Rich text, images, tags  
- **Like & Comment** – Engage with the community  
- **Search & Filter** – Find tours by tags or keywords  
- **Responsive Design** – Works on mobile & desktop
